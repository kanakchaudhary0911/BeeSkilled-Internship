## 📖 Assignments

### Assignment 1 – Linear Search

* Implemented Linear Search using arrays.
* Time Complexity: **O(n)**

### Assignment 2 – Binary Search

* Implemented Binary Search on a sorted array.
* Time Complexity: **O(log n)**

### Assignment 3 – Bubble Sort

* Implemented Bubble Sort with an optimized approach using early termination.
* Time Complexity: **O(n²)**

### Assignment 4 – Reverse Array

* Reversed an array using the two-pointer technique.
* Time Complexity: **O(n)**
